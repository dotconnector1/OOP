Overview Of Template

Support Center Template is best template for Support ticket admin, it is very simple to customize because it uses simple css that has very few line of code. It is based on bootstrap and uses no other heavy plugin that make it very easy to use and customize.

Features List of Template

Support Center Template is featured rich template that are described below:

  • Leight Weight Template
  • Bootstrap 3.1.1
  • Fully Responsive
  • Easy to understand & customize and lots more...
